## Cloudflare Image Hosting

Free unlimited image hosting on Telegraph, deployed on Cloudflare.

### Features

- Free & Unlimited
- Drag & Drop to upload
- Copy & Paste to upload
- Supported formats: image/video/GIF, Max file size is 5MB

### Development

```
npm install
npm run dev
```

### Deployment

```
fork this repo
cloudflare pages -> create a new project -> select your repo
select vue preset -> save and deploy
```
